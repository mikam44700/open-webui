# Avatars des Agents IA

Déposer ici les mascottes 3D (PNG transparent) générées via le "Style DNA" (voir `Avatar.md` à la racine du projet).

Convention de référence dans le code : `/assets/agents/<fichier>.png`
(ex. le template Mike pointe sur `/assets/agents/mike.png`).

Attendu pour Mike : **mike.png** (portrait plan poitrine, fond transparent).
Tant que le fichier est absent, l'UI retombe proprement sur l'emoji 🎼 (pas d'image cassée).
